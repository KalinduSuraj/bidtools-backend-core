# BidTools Backend Core

This is a **NestJS + TypeScript** project built with pnpm.

## Features
- RESTful APIs
- TypeScript support
- Modular architecture
- Easy to extend and maintain

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/my-nest-project.git
```

2. Navigate to the project folder:

```bash
   cd my-nest-project
```

3. Install dependencies using pnpm:

```bash
pnpm install
```

## Running the project

```bash
# development mode with hot reload
pnpm run start:dev

# production mode
pnpm run start:prod
```

The server will run on http://localhost:3000

## Scripts
```bash
pnpm run start → start in production

pnpm run start:dev → start in development

pnpm run build → build the project

pnpm run lint → run linter

```
